# 🛒 Grocery Website Project

## 📌 Project Description
FreshMart Grocery Website is a responsive online grocery shopping platform designed to provide a smooth and user-friendly experience. Users can browse products, view prices, and add items to their cart.

This project is built using HTML, CSS, JavaScript, and Bootstrap, ensuring a modern design and mobile-friendly layout.

---

## 🚀 Features
- Product listing with images and prices  
- Add to cart functionality  
- Dynamic total price calculation  
- Responsive design using Bootstrap  
- Simple and clean user interface  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  
- Bootstrap  

---

## ▶️ How to Run
1. Download or clone the repository  
2. Open `index.html` in any browser
